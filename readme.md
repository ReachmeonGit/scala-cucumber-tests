# Sample Test

## Prerequesits

1. A web browser (Chrome prefered)
2. A Web Driver suitable with your browser
3. Scala Build Tool

<https://www.selenium.dev/documentation/en/webdriver/driver_requirements/>
Setup your webdriver from link above

This Script is compatible with chrome,firefox,edge,IE,opera and safari

Default is chromedriver

## Runing project

Run with default config

``` bash
src/test/resources/AutomationPracticeTestStepsDef.feature
```

this file contains config

### Then run following commands

``` bash
    cd #to root folder
    #open cmd
    sbt # and press enter
    # wait for it to download dependencies and start server.
    # After it is done
    test #and press enter
```

Run with custom browser

```bash
 cd #to root folder
    #open cmd
    sbt # and press enter
    # wait for it to download dependencies and start server.
    # After it is done
    testOnly RunCukesTest -- -Dbrowser=<Browsername>
    # replace <Browsername> with any of the followning
    # chrome,firefox,edge,IE,opera and safari
```

## Working

`Email` and `Password` are generated by faker.
The script is integrated as test for sbt,
sbt tests find cucumber classes and feature files then run tests accordingly

## Report

Html report will be generated in target\cucumber-html-report.html
It will contain full detail of tests

## Failure

Reported failures can be seen in html report file. if a scenario fails a screen shot will be recorded in html report
`target\cucumber-html-report.html`
